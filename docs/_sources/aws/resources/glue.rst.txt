.. _aws.glue-connection.actions.delete:

delete
++++++
Delete a connection from the data catalog

:example:

.. code-block:: yaml

    policies:
      - name: delete-jdbc-connections
        resource: glue-connection
        filters:
          - ConnectionType: JDBC
        actions:
          - type: delete
