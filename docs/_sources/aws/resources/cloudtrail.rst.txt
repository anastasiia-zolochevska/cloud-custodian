.. _aws.cloudtrail.filters.is-shadow:

is-shadow
+++++++++
Identify shadow trails (secondary copies), shadow trails
can't be modified directly, the origin trail needs to be modified.

Shadow trails are created for multi-region trails as well for
organizational trails.

  .. code-block:: yaml

      state:
        type: boolean
      type:
        enum:
        - is-shadow
    required:
    - type


.. _aws.cloudtrail.filters.status:

status
++++++
Filter a cloudtrail by its status.

:Example:

.. code-block:: yaml

    policies:
      - name: cloudtrail-check-status
        resource: aws.cloudtrail
        filters:
        - type: status
          key: IsLogging
          value: False

.. _aws.cloudtrail.actions.set-logging:

set-logging
+++++++++++
Set the logging state of a trail

:Example:

.. code-block:: yaml

  policies:
    - name: cloudtrail-set-active
      resource: aws.cloudtrail
      filters:
       - type: status
         key: IsLogging
         value: False
      actions:
       - type: set-logging
         enabled: True

.. _aws.cloudtrail.actions.update-trail:

update-trail
++++++++++++
Update trail attributes.

:Example:

.. code-block:: yaml

   policies:
     - name: cloudtrail-set-log
       resource: aws.cloudtrail
       filters:
        - or:
          - KmsKeyId: empty
          - LogFileValidationEnabled: false
       actions:
        - type: update-trail
          attributes:
            KmsKeyId: arn:aws:kms:us-west-2:111122223333:key/1234abcd-12ab-34cd-56ef
            EnableLogFileValidation: true

      attributes:
        type: object
      type:
        enum:
        - update-trail
    required:
    - attributes
